# UID-KAT

## Dependencies
It is recommended to run the project with the following main dependencies (or equivalent) for effective development:
- python 3.8
- pytorch 2.1
- cuda 12.1
- cudnn 8.9.2
- timm
- [rational_kat_cu](https://github.com/Adamdad/rational_kat_cu)
- torchinfo
- dominate
- visdom
- opencv
